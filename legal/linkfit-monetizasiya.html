01

Prinsip — Freemium

Əsas oyun loop-u (oyun tapmaq, qoşulmaq, kort tutmaq) pulsuz qalır — bu, böyüməni qidalandırır. Pulu iki yerdən qazanırıq: dərinlik (ciddi oyunçular üçün Premium) və əməliyyatlar (rezervasiya, dərs, turnir).

Niyə əsas loop pulsuz? Pulsuz core = daha çox istifadəçi = daha çox əməliyyat və daha çox potensial Premium abunəçi. Core-u paywall etmək böyüməni öldürür.

02

1. Premium abunəlik — əsas mühərrik

Təkrarlanan, yüksək marjalı gəlir. Ciddi oyunçular abunə olur. Arxasına qoyulanlar:

  • Qabaqcıl statistika — ELO tarixçəsi, qalibiyyət/rəqib analizi, inkişaf qrafikləri (insights modulu artıq hazırdır).
  • Komissiyasız / endirimli rezervasiya — Premium-a keçməyin ən güclü səbəbi.
  • Prioritet giriş — populyar slotlara erkən, daha çox oyun dəvəti.
  • Reklamsız təcrübə.
  • VIP nişanı + profil fərdiləşdirməsi — status, icmada güclü işləyir.

Qiymət (AZ): ~9.90 ₼/ay və ya ~99 ₼/il. Padel premium idmandır — oyunçular ödəmə qabiliyyətindədir.

03

2. Əməliyyat haqları

Rezervasiya

Xidmət haqqı

Hər kort tutuluşundan ~1-2 ₼ və ya 5%. Premium-da pulsuz → Premium-a itələyir.

Dərs

Təlim komissiyası

Məşqçi dərslərindən %10-15 (Öyrən modulu hazırdır).

Turnir

Giriş haqqı payı

Turnir/liqa ödənişindən komissiya.

Etibarlılıq

Gec ləğv / gəlməmə

Kiçik cərimə — həm gəlir, həm etibarlılığı artırır.

04

3-5. Wallet, marketplace və icma gəlirləri

Float

Wallet / kredit

Oyunçu balans yükləyir, rezervasiya/dərsə xərcləyir. Pul səndə qalır + paket satışı (50 ₼ yüklə, 55 ₼ al).

Affiliate

Avadanlıq marketplace

Raketka, top, ayaqqabı — yerli mağaza ilə komissiya və ya birbaşa satış.

İcma

Pulsuz liqalar / mövsüm

Giriş haqqı + mükafat, sən marja saxlayırsan.

Brend

Merch + boost

LinkFit geyimi (gəlir + reklam); tərəf axtaran üçün profil görünürlüyü.

Tövsiyə olunan sıra

  • 1. Premium-u işə sal (qabaqcıl statistika + komissiyasız rezervasiya) — ən təmiz, təkrarlanan gəlir.
  • 2. Yerli ödənişi (Payriff/Epoint) qoş — komissiya + wallet-i açır (ən böyük miqyas).
  • 3. Dərs / turnir komissiyası — marketplace gəliri.
  • 4. Marketplace / merch.

Yekun: əsas loop-u pulsuz saxla, böyü; gəliri Premium + əməliyyatlardan qazan. Ödəniş sistemi qoşulan kimi komissiya və wallet ilə miqyaslan.
